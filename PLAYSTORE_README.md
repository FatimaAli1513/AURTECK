## 📝 Full App Description (4000 characters max)

**Aurteck** is a comprehensive offline-first business management application designed for small business owners, shopkeepers, and entrepreneurs. Manage your business efficiently without internet connection - all your data stays secure on your device.

### 🎯 Key Features

**📝 Memo Management**
- Create unlimited memos with title and description
- Edit existing memos easily
- Real-time search to find memos instantly
- Automatic date tracking for created and updated memos
- Clean card-based interface for easy viewing

**👥 Ledger (Customer Management)**
- Add and manage customer/client information
- Track customer balances with "To Receive" and "To Pay" options
- View total receivables and payables at a glance
- Search customers by name or phone number
- Color-coded balance display (Green for receivables, Red for payables)
- Summary cards showing total amounts to receive and pay

**💰 Expenditure Tracking**
- Record expenses with 8 predefined categories:
  - 🍔 Food
  - 🚗 Transport
  - 🛍️ Shopping
  - 📄 Bills
  - 🎬 Entertainment
  - 🏥 Health
  - 📚 Education
  - 📦 Other
- Category-based organization with color-coded icons
- View total expenditure summary
- Search expenses by title or category
- Add optional descriptions to expenses

**📊 All Records View**
- Unified view of all memos, clients, and expenses
- Filter by type: All, Memos, Clients, or Expenses
- View summary statistics for each category
- See net balance calculation
- Quick navigation to specific screens

**🏠 Home Dashboard**
- Quick action buttons for fast access:
  - New Memo
  - Add Client
  - Add Expense
  - Clear Data (with confirmation)
- Overview statistics showing:
  - Total Memos count
  - Total Clients count
  - Total Expenses count
- Recent activities feed (last 5 activities)
- Pull-to-refresh to update data
- Current date display

## 🎓 How to Use

1. **Adding a Memo:**
   - Tap "New Memo" on home screen
   - Enter title and description
   - Save to create memo

2. **Adding a Client:**
   - Tap "Add Client" on home screen
   - Enter name and phone number
   - Set opening balance (optional)
   - Choose "To Receive" or "To Pay"
   - Save to add client

3. **Adding an Expense:**
   - Tap "Add Expense" on home screen
   - Enter expense title and amount
   - Select category
   - Add description (optional)
   - Save to record expense

4. **Viewing All Records:**
   - Tap "View All" on home screen
   - Use filter tabs to view specific types
   - Tap any record to navigate to details

5. **Searching:**
   - Use search bar on any screen
   - Type to filter results in real-time
   - Search works across titles, descriptions, and categories
